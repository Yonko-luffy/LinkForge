About LinkForge

🎯 Personal URL Namespaces

LinkForge solves the URL collision problem by giving every user their own namespace. Instead of competing for short codes, everyone gets their own space:

  • /alice/project ✅
  • /bob/project ✅
  • /tanish/portfolio ✅

🔄 Dynamic Links

All links are dynamic by default. Change where they redirect anytime without breaking existing shares or bookmarks.

🛠️ Technical Stack

  • Backend: Python Flask with modular blueprints
  • Database: SQLite (upgradeable to PostgreSQL)
  • Frontend: Pure CSS with minimal JavaScript
  • Deployment: Ready for Render, PythonAnywhere, Heroku
